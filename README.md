# FreeBSD port of OpenBSD httpd

- OpenBSD branch is cvs copy
- https://github.com/koue/scripts/blob/master/OpenBSD/cvs_checkout.sh

## Compilation

cd httpd/src/ && make

## Usage

usr.sbin/httpd/httpd -f etc/examples/httpd.conf
